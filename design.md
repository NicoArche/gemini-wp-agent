# Gemini WP-CLI

Un intérprete de lenguaje natural a comandos WP-CLI usando la API de Gemini.

## Descripción

Este proyecto permite convertir instrucciones en lenguaje natural a comandos WP-CLI válidos utilizando la API de Gemini de Google. Incluye un backend en Node.js, un frontend en Vanilla JavaScript y un plugin de WordPress para la ejecución de comandos.

## Stack Tecnológico

- **Backend**: Node.js + Express
- **Frontend**: Vanilla JavaScript + HTML/CSS
- **WordPress Plugin**: PHP
- **API**: Google Gemini API
